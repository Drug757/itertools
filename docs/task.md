# Техническое задание

## 1. Описание задачи
Необходимо разработать простую консольную программу, которая демонстрирует работу библиотеки itertools. Программа должна показывать, как можно генерировать различные варианты данных (комбинации, перестановки и декартово произведение) на основе одного набора значений.

## 2. Цель работы
Цель работы — изучить основные возможности библиотеки itertools и понять, как с её помощью можно упростить перебор данных и генерацию вариантов без использования сложных вложенных циклов.

## 3. Функциональные требования
Программа должна:
- принимать заранее заданный список данных  
- выводить комбинации элементов заданной длины  
- выводить все возможные перестановки элементов  
- выводить декартово произведение элементов  
- отображать результаты в читаемом виде  

## 4. Нефункциональные требования
- программа должна работать быстро за счёт использования итераторов  
- код должен быть понятным и не перегруженным  
- не должно использоваться лишних вычислений и сложных конструкций  
- вывод должен быть структурированным и удобным для чтения  

## 5. Ограничения и допущения
- используется небольшой набор данных (например, список из 3–4 элементов)  
- программа работает только в консоли  
- не используется ввод пользователя  
- результаты выводятся напрямую без сохранения в файл
