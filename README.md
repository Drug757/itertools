# itertools project

## 1. Краткое описание инструмента
itertools — это встроенная библиотека Python для работы с последовательностями и перебором данных.

## 2. Цель изучения
Изучить работу итераторов и научиться использовать готовые инструменты для перебора данных.

## 3. Описание реализованной задачи
Была реализована программа, которая демонстрирует работу основных функций itertools: комбинации, перестановки и декартово произведение.

## 4. Инструкция по запуску
1. Установить Python  
2. Запустить файл:
   ```
   python main.py
   ```

## 5. Общие выводы
Библиотека позволяет писать более короткий и удобный код. Особенно полезна при работе с комбинациями.
